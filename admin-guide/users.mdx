## Overview

User management allows admins to:
- View all user accounts
- Manage user roles
- Handle account issues
- Monitor user activity

## Accessing Users

Navigate to **จัดการผู้ใช้ > ผู้ใช้ (Users > Users)**.

## User List

View all registered users:

| Column | Information |
|--------|-------------|
| **Avatar** | User profile picture |
| **Name** | Display name |
| **Email** | Email address |
| **Role** | User role (user/creator/admin) |
| **Coins** | Current coin balance |
| **Joined** | Registration date |
| **Status** | Active/Suspended |

## Filtering Users

### By Role
- All users
- Regular users
- Creators
- Admins

### By Status
- Active
- Suspended

### By Date
- Registration date range
- Recent activity

### Search
Find users by name or email.

## User Details

Click on a user to view:
- Full profile information
- Account activity
- Coin balance and history
- Purchase history
- PASS status
- Comments made
- Role information

## User Roles

| Role | Capabilities |
|------|--------------|
| **user** | Read content, purchase, comment |
| **creator** | User + publish content |
| **creator_assistance** | Limited creator dashboard |
| **admin** | Admin dashboard access |
| **super_admin** | Full platform control |

## Admin Actions

### View Details
See complete user information.

### Modify Role
Change user role (with appropriate permissions):
- Promote to creator
- Grant admin access
- Revoke special access

<Warning>
Role changes take effect immediately. Only modify roles when authorized.
</Warning>

### Suspend Account
Temporarily disable account access:
- User cannot log in
- Content remains visible
- Can be reversed

### Unsuspend Account
Restore suspended account access.

## User Activity

Monitor user behavior:
- Login history
- Reading activity
- Purchase patterns
- Comment history

## Handling Issues

### Account Recovery
Help users with:
- Password resets
- Email changes
- Access issues

### Suspicious Activity
Investigate:
- Multiple accounts
- Fraudulent behavior
- Policy violations

### User Complaints
Address reports from other users about:
- Harassment
- Inappropriate content
- Policy violations

## User Statistics

Platform-wide user metrics:
- Total registered users
- Active users (daily/monthly)
- New registrations
- User retention

## Privacy Considerations

<Warning>
User data is sensitive. Only access what's necessary for your admin duties.
</Warning>

- Don't share user information
- Access only when needed
- Follow data protection policies
- Document access when required
