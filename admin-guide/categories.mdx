## Overview

Categories help readers find content by genre. Admins can create, edit, and manage categories.

## Accessing Categories

Navigate to **จัดการเนื้อหา > หมวดหมู่ (Content > Categories)**.

## Category List

View all categories:
- Category name
- Description
- Comics using this category
- Created date

## Creating a Category

<Steps>
  <Step title="Click Add New">
    Click **Create Category** or **+ Add**.
  </Step>
  <Step title="Enter Name">
    Provide category name (Thai).
  </Step>
  <Step title="Add Description">
    Optional description for the category.
  </Step>
  <Step title="Save">
    Click **Save** to create.
  </Step>
</Steps>

## Editing Categories

To modify a category:
1. Find in list
2. Click **Edit**
3. Update name or description
4. Save changes

<Info>
Changing a category name updates all comics using it.
</Info>

## Deleting Categories

<Warning>
Before deleting a category:
- Check how many comics use it
- Decide how to handle those comics
- Consider merging into another category instead
</Warning>

To delete:
1. Click **Delete** on category
2. Confirm the action
3. Comics lose that category assignment

## Common Categories

Typical content categories include:

| Category | Description |
|----------|-------------|
| **Action** | Action-packed stories |
| **Romance** | Love and relationships |
| **Fantasy** | Magic and mythical worlds |
| **Comedy** | Humor and laughs |
| **Drama** | Emotional narratives |
| **Horror** | Scary and suspenseful |
| **Sci-Fi** | Science fiction themes |
| **Slice of Life** | Everyday experiences |
| **School** | School settings |
| **Sports** | Athletic stories |

## Category Best Practices

<AccordionGroup>
  <Accordion title="Keep It Simple">
    Don't create too many categories. Keep them distinct and useful.
  </Accordion>

  <Accordion title="Clear Names">
    Category names should be immediately understandable.
  </Accordion>

  <Accordion title="Avoid Overlap">
    Minimize categories that mean the same thing.
  </Accordion>

  <Accordion title="User-Friendly">
    Think about how readers will use categories to find content.
  </Accordion>
</AccordionGroup>
