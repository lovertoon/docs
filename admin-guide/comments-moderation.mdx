## Overview

Comment moderation ensures a healthy community by:
- Removing inappropriate content
- Handling reports
- Maintaining platform standards

## Accessing Comments

Navigate to **จัดการเนื้อหา > ความคิดเห็น (Content > Comments)**.

## Handling Reported Comments

<Steps>
  <Step title="Review Report">
    Read the comment and report reason.
  </Step>
  <Step title="Assess Content">
    Determine if it violates guidelines.
  </Step>
  <Step title="Take Action">
    - Dismiss report (no violation)
    - Remove comment
    - Warn user
    - Additional action if needed
  </Step>
  <Step title="Document">
    Record your decision.
  </Step>
</Steps>

## Moderation Actions

### Remove Comment
Deletes the comment from public view.

### Warn User
Send a warning about the violation.

### Dismiss Report
Mark report as reviewed, no action needed.

### Escalate
Forward to senior admin for complex cases.

## Community Guidelines

Comments that violate:
- Spam or advertising
- Harassment or hate speech
- Explicit inappropriate content
- Personal information sharing
- Threats or incitement
- Copyright violations

## Bulk Moderation

For efficiency:
- Select multiple comments
- Apply action to all
- Process spam waves quickly

## User Patterns

Monitor for:
- Repeat offenders
- Coordinated spam
- Harassment campaigns
- Suspicious activity

## Comment Reports

View report details:
- Who reported
- Report reason
- Comment context
- Similar reports

## Best Practices

<AccordionGroup>
  <Accordion title="Context Matters">
    Consider the full context before removing comments.
  </Accordion>

  <Accordion title="Consistent Standards">
    Apply the same rules to all users equally.
  </Accordion>

  <Accordion title="Timely Action">
    Address reported content promptly.
  </Accordion>

  <Accordion title="Document Decisions">
    Keep records of moderation actions.
  </Accordion>
</AccordionGroup>

## Edge Cases

<AccordionGroup>
  <Accordion title="Criticism vs. Harassment">
    Distinguish between negative feedback (allowed) and harassment (not allowed).
  </Accordion>

  <Accordion title="Spoilers">
    Spoilers may be frustrating but aren't necessarily removable.
  </Accordion>

  <Accordion title="Language Barriers">
    Use translation tools if needed for non-Thai comments.
  </Accordion>
</AccordionGroup>
