## Overview

As an admin, you can view and manage all comics on the platform, regardless of creator.

## Accessing Comics

Navigate to **จัดการเนื้อหา > การ์ตูน (Content > Comics)** in the sidebar.

## Comics List

The list displays:

| Column | Information |
|--------|-------------|
| **Cover** | Comic thumbnail |
| **Title** | Comic name |
| **Creator** | Creator name |
| **Status** | Published/Draft/etc. |
| **Chapters** | Chapter count |
| **Views** | Total views |
| **Created** | Publication date |
| **Actions** | Edit/View options |

## Filtering Comics

### By Status
- All
- Published
- Draft
- Ongoing
- Completed
- Hiatus
- Dropped

### By Content Type
- Comics (การ์ตูน)
- Novels (นิยาย)

### By Adult Rating
- All content
- General only
- Adult only

### By Creator
Search or select specific creator.

## Viewing Comic Details

Click on a comic to see:
- Full information
- All chapters
- Categories
- Creator details
- Performance metrics

## Admin Actions

### Edit Comic
Modify comic details:
- Title and description
- Cover/banner images
- Categories
- Status
- Adult rating

<Warning>
Edit creator content only when necessary. Contact creators before making significant changes.
</Warning>

### Managing Chapters
View and manage all chapters:
- View chapter list
- Edit chapter details
- Remove problematic chapters

### Delete Comic
<Warning>
Deleting a comic is permanent and removes:
- All chapters
- All comments
- All purchase history

This action cannot be undone.
</Warning>

## Content Moderation

### Flagged Content
Review comics flagged for:
- Inappropriate content
- Copyright concerns
- Policy violations

### Taking Action
When content violates policies:
1. Document the violation
2. Contact creator if appropriate
3. Remove or restrict content
4. Follow escalation procedures

## Bulk Operations

<Note>

## Comic Analytics

View platform-wide analytics:
- Most popular comics
