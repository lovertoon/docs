## Accessing the Dashboard

Log in with your admin account and navigate to `/dashboard`.

<Info>
Only users with admin or super_admin roles can access the admin dashboard.
</Info>

## Dashboard Home

The main dashboard displays:

### Platform Statistics

<CardGroup cols={4}>
  <Card title="Total Users" icon="users">
    Registered user count.
  </Card>
  <Card title="Active Creators" icon="pen-nib">
    Approved creators.
  </Card>
  <Card title="Total Comics" icon="book">
    Published content count.
  </Card>
  <Card title="Revenue" icon="coins">
    Platform revenue metrics.
  </Card>
</CardGroup>

### Recent Activity
Quick view of:
- New user registrations
- Recent publications
- Top-up activity
- Creator applications

### Charts & Analytics
Visual representations of:
- User growth over time
- Revenue trends
- Content publication rates

## Navigation Sidebar

### Content Management (จัดการเนื้อหา)

| Menu Item | Purpose |
|-----------|---------|
| **การ์ตูน (Comics)** | View and manage all comics |
| **หมวดหมู่ (Categories)** | Manage genre categories |
| **ความคิดเห็น (Comments)** | Moderate comments |
| **โปรโมท (Promote)** | Manage promotions |
| **โค้ดอ่านฟรี (Free Codes)** | View all free codes |

### User Management (จัดการผู้ใช้)

| Menu Item | Purpose |
|-----------|---------|
| **ผู้ใช้ (Users)** | View and manage users |
| **ครีเอเตอร์ (Creators)** | Manage creator applications |

### Finance (การเงิน)

| Menu Item | Purpose |
|-----------|---------|
| **ประวัติการซื้อ (Orders)** | View all purchases |
| **ประวัติการเติมเงิน (Top-ups)** | View all top-ups |
| **การจ่ายเงิน (Payouts)** | Process creator payouts |

## Notification Badges

Red badges indicate items needing attention:

### Pending Creators
Number of creator applications awaiting review.

### Unread Comments
Comments that need moderation attention.

### Pending Payouts
Scheduled payouts ready for processing.

## Quick Actions

From the dashboard home:
- Jump to pending applications
- View recent activity
- Access key reports
- Handle urgent items

## Search Functionality

Use the search feature to find:
- Specific users
- Comics by title
- Creator accounts
- Transactions

## Filtering & Sorting

Most list views support:
- Date range filtering
- Status filtering
- Sorting options
- Search within results

## Data Export

<Note>
Export functionality may be available for certain data views. Check individual sections for export options.
</Note>

## Mobile Access

The admin dashboard is responsive:
- Works on tablets and phones
- Collapsible sidebar
- Touch-friendly interface
- Full functionality maintained

## Troubleshooting

<AccordionGroup>
  <Accordion title="Can't access dashboard">
    - Verify your account has admin role
    - Try logging out and back in
    - Clear browser cache
    - Contact super admin
  </Accordion>

  <Accordion title="Data not loading">
    - Check internet connection
    - Refresh the page
    - Try a different browser
    - Report persistent issues
  </Accordion>

  <Accordion title="Actions not working">
    - Ensure you have required permissions
    - Check for validation errors
    - Review any error messages
    - Try again or escalate
  </Accordion>
</AccordionGroup>
