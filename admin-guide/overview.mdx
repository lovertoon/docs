## Admin Role

Administrators are responsible for managing the Lovertoon platform:
- Content moderation
- User management
- Creator approval
- Financial oversight
- Platform configuration

<Warning>
Admin features are restricted to authorized personnel only. Misuse may result in access revocation.
</Warning>

## Admin Dashboard

Access the admin dashboard at `/dashboard` with admin credentials.

### Navigation Structure

| Section | Purpose |
|---------|---------|
| **แดชบอร์ด (Dashboard)** | Overview and statistics |
| **จัดการเนื้อหา (Content)** | Comics, categories, comments, promotions |
| **จัดการผู้ใช้ (Users)** | Users and creator management |
| **การเงิน (Finance)** | Orders, top-ups, payouts |

## Admin Levels

### Super Admin
Full platform access:
- All admin features
- System configuration
- Admin management
- No restrictions

### Admin
Standard admin access:
- Content management
- User management
- Financial oversight
- Some restrictions may apply

## Key Responsibilities

### Daily Tasks
- Review pending creator applications
- Moderate reported content
- Process scheduled payouts
- Monitor platform health

### Weekly Tasks
- Review platform analytics
- Approve bidding winners
- Update promotions
- Community management

### As Needed
- Handle support escalations
- Investigate reports
- Update categories
- Special promotions

## Dashboard Badges

The sidebar shows notification badges:

| Badge | Meaning |
|-------|---------|
| **Creators** | Pending creator applications |
| **Comments** | Unread comments (if applicable) |
| **Payouts** | Scheduled payouts awaiting processing |

## Security

<Warning>
Admin access is privileged. Follow security best practices:
- Never share your credentials
- Use strong, unique passwords
- Log out when finished
- Report suspicious activity
</Warning>

## Getting Help

For admin-specific issues:
- Contact platform owner
- Review documentation
- Escalate to super admin
