## What Are Free Reading Codes?

Free reading codes are promotional codes created by creators to give readers free access to their content. These codes:
- Unlock content without spending coins
- Have limited validity periods
- Can only be used once per user

## Getting Codes

Creators distribute codes through:
- Social media (Twitter, Facebook, Instagram)
- Discord communities
- Live events or streams
- Promotional campaigns
- Giveaways and contests

<Info>
Follow your favorite creators to stay updated on code giveaways!
</Info>

## Code Types

### Chapter Code
Unlocks a **specific chapter** in a comic.

### Comic Code
Unlocks **all chapters** of a specific comic (existing and future chapters published during the validity period).

### Creator Code
Unlocks **all content** from a creator (all their comics and chapters).

## How to Redeem a Code

<Steps>
  <Step title="Navigate to Content">
    Go to the comic or chapter you want to unlock.
  </Step>
  <Step title="Click Redeem">
    When the purchase dialog appears, look for:
    - **Redeem Code** button
    - **Use Code** option
    - Code input icon
  </Step>
  <Step title="Enter Code">
    Type or paste the code exactly as provided (codes are case-sensitive).
  </Step>
  <Step title="Submit">
    Click **Redeem** or press Enter.
  </Step>
  <Step title="Enjoy">
    If valid, the content is unlocked immediately!
  </Step>
</Steps>

## Code Validity

### Time Limits

| Duration | Typical Use |
|----------|-------------|
| **1 hour** | Flash promotions |
| **12 hours** | Standard promotions |
| **24 hours** | Event codes |
| **72 hours** | Extended campaigns |

<Warning>
Codes expire based on their creation time, not when you receive them. Redeem codes promptly!
</Warning>

### Usage Limits
- Each code can only be used **once per user**
- Once redeemed, the code becomes invalid for that user
- Different users can use the same code (until it expires)

## Code Statuses

| Status | Meaning |
|--------|---------|
| **Active** | Code is valid and can be redeemed |
| **Used** | You've already redeemed this code |
| **Expired** | Code validity period has ended |
| **Invalid** | Code doesn't exist or is malformed |

## Troubleshooting

<AccordionGroup>
  <Accordion title="Code says 'Invalid'">
    - Check for typos (codes are case-sensitive)
    - Ensure no extra spaces before/after the code
    - Verify you're using the full code
  </Accordion>

  <Accordion title="Code says 'Expired'">
    - The code's validity period has ended
    - Codes can expire even if recently shared (created earlier)
    - Ask the creator for a new code
  </Accordion>

  <Accordion title="Code says 'Already Used'">
    - You've already redeemed this code
    - Each user can only use a code once
    - Check if the content is already unlocked
  </Accordion>

  <Accordion title="Code doesn't work for this content">
    - The code may be for specific content only
    - Check if the code is for the correct comic/chapter
    - Creator codes only work for that creator's content
  </Accordion>
</AccordionGroup>
