## Accessing the Dashboard

Once approved as a creator, access your dashboard at `/dashboard` from:
- Navigation menu
- Profile dropdown
- Direct URL

<Info>
Only approved creators can access the Creator Dashboard.
</Info>

## Dashboard Overview

The main dashboard page shows:
- Quick statistics
- Recent activity
- Performance metrics
- Notifications

## Navigation Sidebar

The sidebar organizes all dashboard features:

### Content Section
| Menu Item | Purpose |
|-----------|---------|
| **การ์ตูน (Comics)** | Manage your comics and chapters |
| **ความคิดเห็น (Comments)** | View and respond to comments |

### Marketing Section
| Menu Item | Purpose |
|-----------|---------|
| **โค้ดอ่านฟรี (Free Codes)** | Create promotional codes |
| **Creator PASS** | Promote as Hot Stories |
| **เสนอราคาโปรโมท (Bidding)** | Bid for promotions |

### Business Section
| Menu Item | Purpose |
|-----------|---------|
| **ประวัติการซื้อ (Orders)** | View chapter purchases |
| **รายได้ (Earnings)** | Track revenue and payouts |
| **ผู้ช่วย (Assistants)** | Manage team members |

### Performance Charts
Visual representations of:
- Revenue over time

## Quick Actions

From the dashboard home, quickly:
- Create a new comic
- Add a chapter
- View earnings
- Check comments

## Navigation Features

### Collapsible Sidebar
Click the collapse button to minimize the sidebar for more workspace.

### User Menu
Access account options:
- View profile
- Sign out
- Settings

### Notifications Badge
Red badges indicate:
- Unread comments (in Comments menu)
- Bidding notifications (in Bidding menu)

## Mobile Navigation

On mobile devices:
- Tap the menu icon to open navigation
- Swipe to dismiss
- Responsive layout for all screens

## Dashboard Sections Explained

### Comics Management
Create and manage your comics:
- Add new comics with details
- Upload chapters
- Set pricing
- Track performance

[Learn more about Creating Comics →](/creator-guide/creating-comics)

### Comments
Engage with readers:
- View all comments
- Reply to readers
- Filter by comic/chapter
- Mark as read

[Learn more about Managing Comments →](/creator-guide/managing-comments)

### Free Codes
Promote your work:
- Generate free access codes
- Set expiration times
- Track code usage
- Share with followers

[Learn more about Free Codes →](/creator-guide/free-codes)

### Creator PASS
Hot Story promotion:
- Feature your comic as Hot Story
- Appear on homepage
- Reach more readers

[Learn more about Creator PASS →](/creator-guide/creator-pass)

### Promotion Bidding
Compete for visibility:
- Bid for banner slots
- Bid for small covers
- Bid for pinned stories
- Weekly bidding periods

[Learn more about Bidding →](/creator-guide/promotion-bidding)

### Orders
Track purchases:
- See who bought chapters
- View transaction details
- Export for records

### Earnings
Manage finances:
- Track accumulated earnings
- View payout history
- Request payouts
- Set payout preferences

[Learn more about Earnings →](/creator-guide/earnings-overview)

### Assistants
Team management:
- Invite assistants
- Set permissions
- Remove team members

[Learn more about Assistants →](/creator-guide/assistants)
